# Todo Capsule

Todo Capsule 是一个 macOS 桌面边缘待办胶囊。常驻屏幕边缘可快速调取，适合工作时快速记一条、扫一眼、勾掉。
接入AI模型后，可快速结合一定周期内的已完成和进行中的待办项，快速生成日报、周报、绩效总结等内容。

## 下载安装

下载最新版 DMG：

[todo-capsule-v1.0.9.dmg](https://github.com/Seth360/todo-capsule/releases/download/v1.0.9/todo-capsule-v1.0.9.dmg)

所有版本见 [GitHub Releases](https://github.com/Seth360/todo-capsule/releases)。

提示：当前安装包未做 Apple Developer ID 公证。首次打开如果被 macOS 拦截，请右键 App 选择“打开”。

## 功能

- 全局热键快速记录待办
- 小窗一瞥、输入态、大窗面板三种主要工作形态
- 清单、标签、收藏夹、回收箱
- 行内编辑、拖拽排序、置顶、撤销
- 亮色/暗色/跟随系统主题
- AI 总结配置与生成总结的模版设置

## 运行

```bash
cd code
swift run
```

要求 macOS 14+，并安装 Xcode 或 Command Line Tools。
