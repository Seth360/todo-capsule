# Todo Capsule 设计规范

本规范是 Todo Capsule 后续界面实现的唯一基础规则；新增页面、组件和状态都应复用本文的令牌，不得再单独定义蓝色主操作。

## 1. 视觉基调与颜色

- 深色画布：`#1C1C1C`；侧栏：`#191919`；浮层与输入表面：`#24272B`。
- 主操作仅用绿色：`#0B9153`，按下为 `#087A46`。蓝色不作为按钮、头像、输入焦点或选中背景。
- 主文字：`#F2F3F5`；次级文字：`#AAAFB7`；弱提示：`#727780`；深色分割线：白色 10% 透明度。
- 选中项使用中性白色 10% 透明底；图标和文字只在选中、完成或成功状态使用绿色。

## 2. 字体与层级

- 字体：Inter，缺失时使用系统无衬线字体。
- 页面标题 22/semibold；弹窗标题 20/semibold；正文 14/regular；列表和按钮 13/medium；分组标题与辅助信息 11–12。
- 同级分组标题保持同一字号、字重和弱文字色；分组标题不加图标。

## 3. 间距、圆角与边框

- 采用 4pt 网格：4、8、12、16、24、28。页面横向内边距 28，弹窗内边距 28，控件间距 8 或 12。
- 控件圆角 8；输入框圆角 10；弹窗圆角 16。
- 输入框、下拉框和次级按钮均为透明或中性表面 + 1px 低对比边框；不使用蓝色底色。

## 4. 基础组件

- 主按钮：绿色实底、白色文字，最小高度 32。仅用于确认、保存、生成等唯一主动作。
- 次级按钮：中性低对比底色与边框，用于取消、关闭、选择模板等。
- 输入框：透明/深灰表面、1px 边框、左右 12px 内边距；焦点只提高绿色边框或描边，不填充蓝色。
- 下拉按钮：作为独立次级按钮呈现，文字 + 向下箭头；不伪装成输入框。
- 弹窗：标题、说明、控件、操作区按 16–24px 垂直节奏排列；操作区右对齐，取消在左、绿色确认在右。

## 5. 导航与交互状态

- 目录图标默认使用次级文字色，命中后才变为绿色；列表/标签的更多操作仅在悬停该行时显示。
- 清单、标签分组之间必须有分隔线；标题无图标。将鼠标悬停在整个分组区域时，标题右侧才显示新增加号。
- 智能总结初始只显示记录列表；用户点选记录后，详情栏从右侧滑入。
- 设置页使用无卡片背景的行式布局；不使用 grouped Form 的大色块或额外卡片内边距。
